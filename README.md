# Força Bruta, Backtracking e Recursão em C++

Projeto desenvolvido durante os estudos de Algoritmos e Estruturas de Dados utilizando técnicas de Força Bruta, Backtracking e Recursão.

## Funcionalidades

- Maximum Subsequence Sum
- Subset Sum
- Sales Trip
- Resolução de Labirinto
- Busca Recursiva
- Backtracking

## Conceitos Trabalhados

- Força Bruta (Brute Force)
- Backtracking
- Recursão
- Busca em Espaço de Estados
- Análise de Complexidade
- Exploração de Caminhos

## Estrutura do Projeto

### subseqmax.cpp

Implementação da busca da subsequência de soma máxima utilizando:

- Força Bruta
- Recursão

### subsetsum.cpp

Implementação do problema Subset Sum utilizando:

- Força Bruta
- Backtracking

### sales_trip.cpp

Determinação do maior lucro possível em uma sequência de viagens.

### labirinto.cpp

Resolução de labirintos utilizando busca recursiva e backtracking.

## Tecnologias

- C++
- STL
- Vector
- Recursão
